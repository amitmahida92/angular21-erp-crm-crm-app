# crm-app

CRM remote for leads, opportunities, customers, and a pipeline board placeholder.

## Federation

Exposes:

```text
./Routes -> projects/crm-app/src/app/remote-entry.routes.ts
```

The shell mounts this remote at `/crm`.

## Commands

```bash
npm run start:crm
npm run build:crm
```

Standalone URL: `http://localhost:4201`
